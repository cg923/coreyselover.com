Zorbit's Orbits

Zorbit's Orbits is a 2D platforming game released by Happyrock Studios and inspired by games like Sonic the Hedgehog, but with a Mario Galaxy twist. The game is written in C++ utilizing the Box2D and SFML libraries, as well as a custom XML-like markup language. www.zorbitsorbits.com
